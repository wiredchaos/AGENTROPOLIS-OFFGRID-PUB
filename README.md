# AGENTROPOLIS OFFGRID PUB

## The Intelligence Grid

> Infrastructure becomes the terrain others must build on.

Agentropolis Offgrid Pub is the public, push-ready repository scaffold for the Agentropolis Intelligence Grid.

Agentropolis is not a landing page. It is an intelligence substrate organized into three layers:

1. **Infrastructure** — runtime, memory, skill registry, dispatch, policy, audit.
2. **Districts / Institutions** — domain-owning zones that publish and govern Skills.
3. **Applications** — user-facing surfaces that consume Skills through the Grid.

Districts publish Skills. Applications consume Skills. The Grid routes, audits, and governs the work.

## Repository Status

This repo is designed as a public bootstrap foundation. It contains architecture docs, district specs, skill registry examples, schemas, and a minimal TypeScript runtime scaffold.

## Core Packages

| Package | Role |
|---|---|
| `agent-runtime` | Executes bounded agent tasks under policy constraints. |
| `skill-registry` | Stores skill contracts, metadata, triggers, and chain maps. |
| `dispatch-protocol` | Routes requests to the correct district and skill. |
| `memory-layer` | Handles structured memory events and audit-safe recall. |
