# Contributing

Agentropolis contributions should strengthen the Grid.

## Contribution Rules

- Add skills as contracts, not vague prompts.
- Keep districts domain-specific.
- Do not bypass policy, audit, or receipt requirements.
- Never commit secrets.
- Include examples for new protocols.

## Pull Request Checklist

- [ ] Skill or protocol has a clear role.
- [ ] Triggers are natural language and specific.
- [ ] Required dependencies are listed.
- [ ] Risk and execution boundaries are documented.
- [ ] Registry check passes.
